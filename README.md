
# Prediksi Biaya Asuransi

Aplikasi Streamlit untuk memprediksi biaya asuransi menggunakan model Regresi Linear Berganda.

## Fitur yang digunakan

- age
- bmi
- children
- sex_male
- smoker_yes
- region_northwest
- region_southeast
- region_southwest

## Model

Regresi Linear Berganda

R² Test = 0.8047
